# stormbuf 周刊 ( 2 ) ：记录下我当前在 Mac OS 上使用的应用

# 使用的应用以及介绍

  1. 开发相关
    1. idea
    2. vscode
    3. docker
    4. postman
    5. jmeter
    6. Xcode
    7. dbleaver
    8. Another Redis Desktop Manager
  2. 滴答清单。用于备忘录，看板，tips 记录
  3. 向日葵远程控制
  4. Bartender 4。用于管理菜单栏图标，主要还是用来隐藏对我来说用不上的图标。这个功能没有系统自带，需要第三方提供也是我没想到的
  5. Amphetamine。屏幕熄屏后会断网，但我需要在下载东西时，熄屏也不会断网，这个应用帮我解决了这个问题。
  6. Downie 4。下载神器
  7. Fluent Reader. RSS 阅读器，使用 win 系统时就是用这个阅读器，没想到 Mac 系统也有这个，这个阅读器已经能满足我的需求了，所以继续使用。
  8. HapiGo。启动器，在使用 win 时，用的是 utools ，虽然 utools 的插件很丰富，但很多插件都欠缺维护，所以想换一款使用试试。体验过 Alfred，但在知道 Raycast 和 HapiGo 后，我还是更喜欢颜值高的。Raycast 也很强大，但最后还是选择对中文和拼音更友好的 HapiGo。
  9. iShot。截图+长截图+录屏。
  10. iterm 2。终端，iterm+oh-my-zsh 的组合真的很赞。
  11. Homebrew。包管理器。
  12. Parallels Desktop. pd + win11，虽然切换到了 Mac OS，但时不时还是想玩两把喜加一的单机。正在寻找可以在 Mac OS 上游玩的单机。
  13. MacZip
  14. IINA。播放神器。
  15. 超级右键。
  16. APPCleaner。卸载应用以及清除残留 ( 重要！ )
  17. 腾讯柠檬。管理启动项+系统状态监控。
  18. PicGo。图床上传。
  19. SwitchHosts。管理本地 host 文件。
  20. Typora
  21. wps
  22. BeMyEars。为讲话声音输出字幕。
  23. 东方财富。做慈善。
